## Project Description
This project analyzes the open-source software Git. It covers its origin, license, Linux usage, ecosystem, and comparison with proprietary tools. It also includes five shell scripts demonstrating Linux concepts.

---

## Scripts Overview

### Script 1 — System Identity Report
Displays system information like kernel version, user, uptime, and date.

### Script 2 — FOSS Package Inspector
Checks if Git is installed and displays its details.

### Script 3 — Disk and Permission Auditor
Shows permissions and size of important directories.

### Script 4 — Log File Analyzer
Counts occurrences of a keyword in a log file and shows matching lines.

### Script 5 — Open Source Manifesto Generator
Generates a personalized manifesto based on user input.
